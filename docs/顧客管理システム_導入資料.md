# 治療院向け顧客管理システム 導入検討資料

## 1. システム概要

治療院に特化した顧客管理システムです。患者情報の一元管理、売上分析、回数券管理、地域分布分析など、治療院経営に必要な機能を網羅しています。

### 主要機能一覧

| 機能 | 内容 |
|------|------|
| 患者管理 | 氏名・住所・主訴・既往歴・来院経路・職業など詳細登録。AI音声入力対応 |
| 伝票管理 | 来院ごとの施術メニュー・金額・支払方法を記録。一括入力・CSV取り込み対応 |
| 回数券管理 | 15/30/45回券+カスタム。購入日・消費開始日・有効期限・消費ペース表示 |
| 売上分析 | 売上推移・新規vs既存・リピート率・LTV分析・客単価・時間帯別分析 |
| 地域分布 | 患者の地図表示（マップ/ヒートマップ/テーブル）、商圏分析 |
| 離反アラート | 30日以上来院がない患者を自動検出、ホーム画面に通知 |
| AI分析 | AIによる経営アドバイス、CSV音声入力での自動データ解析 |
| 月間統計 | 月別の売上・新規数・リピート率・LTV・CPA・稼働率を自動集計 |
| ROAS分析 | 広告費用対効果の分析、媒体別の費用追跡 |
| マスター管理 | メニュー・スタッフ・症状・休日などの基本データ管理 |

### 技術仕様

| 項目 | 内容 |
|------|------|
| フロントエンド | Next.js（React）、レスポンシブ対応（スマホ/PC） |
| データベース | Supabase（PostgreSQL）、SSL暗号化通信 |
| 認証 | Supabase Auth（メール+パスワード） |
| データ分離 | RLS（Row Level Security）による治療院ごとの完全なデータ分離 |
| ホスティング | Vercel（99.9% SLA）、CDN配信 |
| AI | Claude API（音声入力の解析、経営アドバイス） |
| 決済 | Stripe（サブスクリプション管理） |

### 5つの強み

**1. 圧倒的な低価格**
月額4,980円は競合の1/2〜1/3。買い切り49,800円（年間換算4,150円/月）で永続利用も可能。

**2. AI搭載で入力の手間を削減**
音声やテキストをAIが自動解析。患者カルテの入力、来院データの一括取り込みがスマホから30秒で完了。

**3. 治療院に特化した分析機能**
LTV・リピート率・離反予測・ROAS・地域分布など、治療院経営に本当に必要な指標だけを自動集計。汎用CRMにはない分析が標準搭載。

**4. 回数券の消費ペース管理**
購入日と消費開始日を分離でき、消費ペースをリアルタイム表示。「順調」「遅れ気味」が一目で分かるので、次回購入の案内タイミングを逃さない。

**5. 初期費用ゼロ・縛りなし**
初期費用0円、最低契約期間なし。いつでも解約可能。データエクスポートも対応しているのでロックインの心配なし。

### データ移行サポート

既存システムからの移行もサポートしています。

1. 患者データCSVを用意（名前・連絡先・住所等）
2. 来院履歴CSVを用意（日付・メニュー・金額等）
3. AIが自動でカラムマッピング → 一括取り込み
4. 所要時間: 約30分（500件の場合）
